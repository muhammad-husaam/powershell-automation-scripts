## 📖 Overview

This repository demonstrates **real-world PowerShell automation tasks** used in IT support environments to reduce manual effort, improve efficiency, and support system administration tasks.

Each script includes:
- Clear purpose and use case  
- Practical commands used in IT environments  
- Screenshots showing execution and output  

---

## 📌 Purpose

To showcase hands-on PowerShell skills relevant to:
- Help Desk Support
- System Administration
- IT Operations

---

## 🧰 Scripts

| Script | Purpose | File |
|---|---|---|
| System Information Report | Displays OS, RAM, and system details | [system-info.ps1](scripts/system-info.ps1) |
| Disk Space Monitor | Checks disk space and alerts if low | [disk-space-check.ps1](scripts/disk-space-check.ps1) |
| Service Status Checker | Lists running/stopped services | [service-status.ps1](scripts/service-status.ps1) |
| Create Active Directory User | Automates new user creation in AD | [create-ad-user.ps1](scripts/create-ad-user.ps1) |
| User Account Audit | Lists local user accounts | [user-audit.ps1](scripts/user-audit.ps1) |
| Backup Files | Automates file backup to a network location | [backup-files.ps1](scripts/backup-files.ps1) |
