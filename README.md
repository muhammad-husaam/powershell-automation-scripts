# PowerShell Automation Scripts for IT Support

**Author:** Muhammad Husaam  
**Last Updated:** 2026

---

## 📖 Overview

This repository contains **PowerShell scripts** designed to **automate common IT support tasks**.  
Each script includes:

- Clear description of the task  
- Step-by-step instructions  
- Screenshots showing output/results  

The goal is to **demonstrate practical automation skills** that reduce manual effort and improve efficiency.

---

## 🧰 Scripts

| Script | Purpose | File |
|---|---|---|
| Create Active Directory User | Automates new user creation in AD | [create-ad-user.ps1](scripts/create-ad-user.ps1) |
| Monitor Disk Space | Checks disk space and alerts if low | [monitor-disk-space.ps1](scripts/monitor-disk-space.ps1) |
| Backup Files | Automates file backup to a network location | [backup-files.ps1](scripts/backup-files.ps1) |
